# 《星空入侵者》关卡敌人与弹幕配置文档

## 一、系统概述
游戏共设计3个关卡，每个关卡包含多波小怪、1个道中精英/Boss、1个关底Boss。所有配置数据集中管理在`js/config/level.js`中，支持快速调整难度和玩法。

## 四、弹幕系统说明

### 基础弹道类型
| 弹道类型 | 说明 | 特性 |
|---------|------|------|
| `straight` | 直线弹道 | 匀速直线飞行，最基础类型 |
| `curve` | 曲线/弧形弹道 | 角度随时间变化，形成弧形轨迹 |
| `spiral` | 螺旋弹道 | 角度持续旋转，同时加速/减速飞行 |
| `homing` | 追踪弹道 | 自动朝向玩家位置平滑转向，最大转向角度0.05弧度/帧 |

### 弹幕生成工具方法
- `createSector()`：生成扇形弹幕，支持自定义数量、角度范围、起始角度
- `createRing()`：生成环形弹幕，360°均匀分布
- `createSpiral()`：生成螺旋弹幕，支持自定义角速度

## 五、配置调整指南
### 调整敌人难度
1. 修改`ENEMY_CONFIG`中的对应敌机属性：`hp`（血量）、`speed`（速度）、`shootInterval`（射击间隔）
2. 调整`LEVEL_CONFIG`中波次的敌人数量和组合

### 调整Boss难度
1. 修改`BOSS_CONFIG`中的`hp`（总血量）、`phaseCount`（阶段数）
2. 调整各阶段的`shootInterval`（射击间隔）、`shootPatterns`（弹幕模式组合）
3. 新增/修改`shoot()`方法中的弹幕模式实现

### 新增弹幕模式
1. 在`Boss.shoot()`方法中添加新的case分支
2. 可以组合使用现有弹道类型和生成方法
3. 复杂弹道可以通过组合多个基础弹幕实现
