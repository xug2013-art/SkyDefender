# 《星空入侵者》快速开发指南

## 1. 环境准备
### 1.1 开发工具
- 下载并安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 支持Node.js 14+版本（用于eslint代码检查）

### 1.2 项目导入
1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择项目根目录 `d:\workspace\SkyDefender`
4. 项目类型选择"小游戏"
5. 点击"确定"完成导入

## 2. 快速上手

### 2.1 目录结构说明
```
SkyDefender/
├── docs/                # 文档目录
├── js/                  # 游戏源代码
├── images/              # 游戏图片资源
├── audio/               # 游戏音效资源
├── game.js              # 微信小游戏入口
├── game.json            # 微信小游戏配置
└── project.config.json  # 项目配置
```

### 2.2 核心文件修改指引

#### 2.2.1 修改游戏配置
修改 `js/config/level.js` 文件调整游戏难度、敌人属性、关卡配置：
```javascript
// 修改敌机生成间隔
export const ENEMY_GENERATE_INTERVAL = 60 // 单位：帧，越小生成越快

// 修改敌机属性
export const ENEMY_TYPES = {
  'small': {
    hp: 1, // 生命值
    speed: 2, // 移动速度
    score: 10, // 击杀得分
    // ... 其他属性
  }
}
```

#### 2.2.2 新增敌机类型
1. 在 `js/config/level.js` 中新增敌机配置
2. 在 `js/npc/enemy.js` 中添加对应的行为逻辑（如果需要）
3. 在 `js/main.js` 的敌机生成逻辑中添加对应类型的生成

#### 2.2.3 新增弹幕类型
1. 在 `js/bullet/danmaku.js` 中新增弹幕弹道逻辑
2. 在敌机配置中指定使用新的弹幕类型

#### 2.2.4 新增道具
1. 在 `js/prop/index.js` 中新增道具配置
2. 添加道具效果逻辑
3. 配置道具掉落率和合成配方

#### 2.2.5 修改UI界面
修改 `js/runtime/gameinfo.js` 文件调整UI界面：
- 修改开始菜单
- 修改游戏HUD显示
- 修改结束界面
- 添加新的交互按钮

### 2.3 调试技巧
1. **真机调试**：微信开发者工具中点击"真机调试"，扫码在手机上调试
2. **日志输出**：使用 `console.log()` 输出日志，在开发者工具控制台查看
3. **性能面板**：使用微信开发者工具的性能面板查看帧率和内存使用情况
4. **断点调试**：在Sources面板中打断点调试代码

## 3. 常见问题

### 3.1 游戏运行卡顿
- 检查是否有大量对象没有回收到对象池
- 检查是否有内存泄漏
- 减少同一帧内的运算量
- 优化碰撞检测逻辑

### 3.2 音效不播放
- 检查音效文件路径是否正确
- 检查微信小游戏的音效权限配置
- 确保音效文件格式正确（推荐使用mp3格式）

### 3.3 真机上图片不显示
- 检查图片路径是否正确
- 检查图片大小是否超过微信限制（单张不超过2M）
- 检查图片格式是否支持（推荐使用png/jpg格式）

### 3.4 提交审核被拒
- 检查是否包含敏感内容
- 检查是否遵守微信小游戏平台规范
- 确保游戏内容符合相关法律法规

### 4.1 新增关卡

### 4.2 新增玩家技能

### 4.3 新增成就系统
1. 设计成就条件和奖励
2. 添加成就状态存储（使用微信小游戏Storage API）
3. 添加成就UI界面

### 4.4 新增排行榜
1. 接入微信小游戏开放数据域
2. 实现排行榜数据上报
3. 实现排行榜UI展示

## 5. 代码规范
- 使用ES6+语法
- 变量和函数使用小驼峰命名法
- 类名使用大驼峰命名法
- 常量使用全大写下划线分隔
- 提交代码前执行 `eslint js/**/*.js` 检查代码
